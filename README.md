# SnowbreakToolbox

简体中文 | [English](Document/README_en.md)

尘白禁区小工具，目前包括启动器、抽卡记录统计、宿舍拼图功能

---

## 运行环境需求

- .Net 8.0 Runtime

- 管理员权限

---

## 使用

![loading-ag-1133](Document/Images/Display.png)



### 启动

- 提示选择游戏路径时，选择 `启动器所在文件夹`

- 若为Steam端游戏，在设置中设置 `游戏平台` 为 `Steam`



### 记录统计

- 刷新前在程序中手动选择当前记录对应卡池



### 宿舍拼图

- 手动设置数量

- 右侧可切换是否必选此类型

- 点击格子可设置是否可用
